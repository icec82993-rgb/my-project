## 📂 已上线核心项目展示 (Production Projects)

### 📌 项目一：个人全栈修行响应式主页 / 简历
* **技术亮点**：精细化 HTML5 语义化标签布局，重构 CSS3 响应式栅格系统，完美适配全终端大屏与移动端。
* **交付成果**：作为全栈修行的开篇之作，实现了现代审美布局与极致的渲染加载优化。

### 📌 项目二：企业级 RAG 知识库调试看板 (vue3-rag-knowledgebase)
* **技术亮点**：
  * **真·API流式接入**：规避第三方臃肿 SDK，基于原生 Web Fetch API 与 `ReadableStream` 逐步读取二进制流，完美攻克 DeepSeek 大模型 SSE（Server-Sent Events）逐字流式吐字响应与低延迟渲染。
  * **RAG 管道仿真实现**：利用 Vue 3 的 `computed` 响应式数据流，自研微秒级多关键词匹配检索器，实现私有知识切片（Chunks）的动态召回、相似度评分（Cosine Score）计算与 System Prompt 上下文的自适应增量编排。
  * **组件化重构**：采用标准大厂 SFC 规范，构建硬核科技感全屏暗黑视觉（Sci-Fi Dark Console），自主编写轻量级正则 Markdown 解析器，完成高频吞吐下的丝滑 DOM 滚动追光。
